約制國民黨大怪獸

有個人卓越條件、雙李相挺、三隻小豬旋風和馬英九無能的爛攤子，蔡英文仍未勝選，凸顯國民黨這隻大怪獸的力道和加以約制的必要：它有錢、有組織，為達目的不擇手段。 蔡英文造勢場子的聲勢驚人，氣氛感人，但卻是空氣票，畢竟不敵國民黨從威權時代延續下來，平日小恩小惠、耳語洗腦的網絡動員，和對特定選民的政策買票。 宋楚瑜三戰三敗：首戰挾省長撒錢的人情，贏連戰而輸陳水扁；二戰淪為連戰老二，運用國民黨金錢及組織，挑戰陳水扁也落敗。這次不甘被併吞，卻無袖難舞，知其不可為而為之，只能創下比連戰多一敗的紀錄。 民進黨這次有形象清新、學識能力優越的候選人，仍挑戰不成，令人扼腕，只能再等下次機會。 四年後大選，誰繼馬英九登場，將考驗國民黨。吳敦義既為備胎，視接班為當然，但他形象不佳，既無「光環」，又非「正統」，未必見容於國民黨的新貴族。 民進黨只贏過兩次總統大選，一次是贏「空席」，一次是保連任；兩黨都還沒有打敗過尋求連任者之紀錄。因此，如果候選人條件傑出，民進黨下次勝選的機會比較高。 在執政機會再來之前，民進黨應加強組織和培養人才，尋求國會實力成長。這次選舉民進黨立委席次增加，和國民黨一些畸形政客敗選，逐漸突出民主的淘汰功能。 人民作出選擇，民進黨雖未能重返執政，但逾六百萬選票的能量，和四十席立法委員的實力，絕非馬英九、中國或美國所能漠視，在未來四年要善用國會，約制怪獸：打擊濫權，衛護民主與民生，也捍衛台灣主權。（作者王景弘，資深新聞工作者）
January 17, 2012

老闆當卒子

在自由市場經濟體制下，企業老闆對營運及人事有拍板權，員工爭相奉承，政客對他們也要讓三分。享有這種威風的人，不會願意當卒子聽人使喚。 但台灣的大企業家卻走衰運，因為他們想在非自由市場體制的中國享受特權經營，而作了人家的政治卒子，受到政治惡棍的擺佈。 這些有錢大亨忘本，甘當「胡前卒」與「馬前卒」，竟恐嚇台灣人民，要接受無中生有的「九二共識」，一個中國原則，讓台灣成「中國的一部分」。 本土政權執政時，堅持台灣主權獨立，企業家被中國綁架，「以商逼官」。許文龍被迫反扁，卻故意一字不易發表中國的聲明稿，以暴露其荒謬不可信；張榮發因私誼挺扁，立遭北京打壓，轉當統派馬前卒。 復辟的國民黨政權，不用商人逼，已屈降中國條件，只是台灣人民不依，北京便與馬政權聯手，指使大企業老闆集體白賊，「以商逼民」，要保馬特首權位。 專制體制國家，獨裁者靠說謊粉飾太平；人民靠說謊以免牢獄之災。中國就是說謊大國，在共黨統治下，說謊已成中國人專長。 兩蔣戒嚴統治，國民黨宣傳「自由民主」勝過中國，誇稱台灣雖沒有說話的自由，卻還有「不說話的自由」；但馬政權獨裁學中國，「不說話的自由」正在消失，說謊成求生術。 「九二共識」，馬英九帶頭要政府說謊，企業老闆跟著說謊，還要全民說謊。這些大亨在中國創造就業機會、向中國政府納稅，但無力影響中國政府，卻要逼台灣人民接受中國的勒索，這是什麼觀念？ 大老闆說謊，聲音雖大，但人民可以證明：誠實的選票，聲音更大。 （作者王景弘，資深新聞工作者）

星期專論／堂堂正正作台灣的主人

現代民主體制，人民是國家的主人，合格選民在大選中票票等值，可以自由、獨立的選擇自己能信任的候選人，撤換不稱職的笨蛋或壞蛋。以集體意志選擇政治領袖，這就是民主選舉的基本功能。 台灣人民在十四日大選將有機會選擇總統和立法委員，也選擇自己所期待的未來；合格選民應該善用神聖的選票，作一個堂堂正正的台灣主人。 在台灣民主的坎坷進程上，這次選舉具關鍵性意義。如果選民一本良知，拉下已證明無能、犧牲台灣主權和違法亂紀的國民黨，台灣民主將向理想邁進一大步；否則台灣民主、人權將繼續倒退，並有被中國併吞的立即危險。 台灣民主深受美國啟發；美國有長期民主傳統，選民能獨立判斷，自由作主，不會受威迫利誘，被各國視為民主典範。美國民主制度不斷完善，政黨的活動和競選經費的籌措都有嚴密規範，各政黨在共同規範下公平競爭。 反觀台灣，國民黨曾長期一黨專政，擁有龐大黨產，掌控盤據在政府和社會各階層的黨員，習於欺騙、腐化、賄選和恐嚇選民，破壞自由公平選舉。 馬英九黨產「歸零」的承諾，證明是謊言；國民黨是阻礙台灣政治自由、公平競爭的禍根。國民黨自己不會放棄不公不義的既得利益，這需要選民用選票打敗黨國怪獸，使它脫胎換骨，台灣民主才有公平競爭機會。 以美國選民的投票行為而言，不論是依執政成敗，或看候選人的個人條件，這次大選對台灣未來最有利的選擇只有一個：再次政黨輪替，撤換無能、失敗的馬英九政權，選出蔡英文當總統，國會讓民進黨過半。 美國選民投票，第一優先考慮是經濟問題，而失業率與通貨膨脹率是最重要指標。雷根和柯林頓尋求連任成功，因為他們第一任經濟情勢看好，選民也就不在乎雷根年紀大，或柯林頓有寡人之疾。卡特與老布希連任失敗，因為任內經濟不景氣，通貨膨脹，失業率高；卡特道德再高尚，老布希再怎麼老好人都要被開革。 馬英九把台灣經濟奇蹟弄成台灣之恥，失業率高，工作機會流失，薪資倒退，所得降低，物價、房價高漲，貧富差距擴大。他的經濟策略完全依賴中國，但中國既非民主法治，也非自由經濟，問題重重；台商投資幫中國創造就業機會，台灣卻要承受失業和薪資降低的後果。 外交政策成敗是影響美國大選的第二大因素：伊朗革命份子長期佔據美國大使館，卡特政府既無力營救，又無法解決，美國威信大失。經濟與外交雙重無能與失敗，注定他連任失敗的命運。 馬英九政府自棄國格，外交停擺，悉聽中國使喚，而且不受國會監督，不經人民同意，在事涉國家主權的問題上擅自與中國達成秘密協議，還以謊言把責任栽贓前總統李登輝。 他否認犧牲台灣主權，但台灣絕大多數人的國家定位是中華民國為台灣現行國號，台灣並不是中國的一部份，這與馬英九接受虛構的「九二共識」一個中國原則，台灣是中國一部份，絕不相同。 他把台灣定位為「家園」，只有包括全中國大陸的虛構中華民國才是「國家」，那是無視台灣人民意志的流亡者過客心態。他在二○○八年大選時用以偽裝的人民自決誓言和執政後露出的真面目，只凸顯他的誠信問題嚴重。 不只綠營指控馬英九出賣台灣，國民黨大老在維基解密的外交密房話，也提供許多馬英九背叛台灣的佐證。上次大選，馬英九高喊「愛台灣」，認同台灣，台灣前途要由二千三百萬人民決定。但蕭萬長告訴楊甦棣，這是國民黨的選舉策略：要把國家認同問題混淆成與民進黨「無異」，以便放手打經濟牌和反貪腐牌。 但馬英九並非認同台灣；連戰就告訴楊甦棣：「馬英九認同的是中華民國，不是台灣。」對馬英九而言，這兩者並非等同，他的「中華民國」仍然是一九四五年或一九四九年佔領台灣的「中華民國」。他無視台灣民主演變，自己解讀「憲法」，死抱「一個中國」，接受「九二共識」。 蘇起剛接國安會秘書長時，美國在台協會曾向華府報告他的個人背景資料，對所謂「九二共識」有明確的陳述：蘇起在二○○五年出版的書中承認這是他所杜撰，希望陳水扁政府做為兩岸恢復協商的基礎，「但民進黨政府不接受；國民黨（馬英九）新政府與中華人民共和國則接受九二共識為對話基礎。」 馬英九與中國簽下經濟協議，達成出席世衛大會諒解，都以「九二共識」的「一個中國」為基礎；王金平抱怨經濟協議不送立法院審議，不尊重國會，而出席世衛大會付出的代價竟是官方文件出現「中國台灣省」。 連戰曾向楊甦棣坦言，北京應瞭解兩岸任何共識都必須台灣人民所能接受；胡錦濤知道，台灣政治領袖如果想保住他們的位子，必須主張台灣維持不同的政治身分。 現在馬英九想保住他的位子，卻要台灣人民吞下他獨裁接受中國的一個中國條件，而且要栽贓別人，台灣選民能接受嗎？王金平說陳雲林訪問台灣引起的反應，對中國及馬英九都是一次震撼，他們都應該知道不能急推「統一」。但馬英九不是在急推會立即改變現狀的「和平協議」嗎？ 馬英九執政近四年，浮華浪費，經濟困頓；台灣主權流失，法律、人權、民主被踐踏；他學中國復辟黨國體制，肆無忌憚違法濫權，變造官文書意圖栽贓蔡英文，並以國家安全機構監控選情，動用政府資源輔選。這是一個無能、不公正、不能信賴、自絕於人民的政府。已經被騙過一次的台灣選民，沒有理由再被騙第二次。（作者王景弘，資深新聞工作者）

老馬重操舊業

聽其言，觀其行，馬英九絕對不是「表裡如一」的人。凡事雙重標準，玩弄文字定義規避責任，他的問題就在口是心非，說一套做一套，被人看破手腳。 其人言行不一，誰都可以舉出一籮筐，上自國家主權、違法濫權栽贓與情治監控選舉，下至唯一「正面」手段「荷馬史黛」都是如此。 「荷馬史黛」是上次大選「郎史黛」的重演，英文不通且先不說，他不敢用漢字表達，問題就怕穿幫：兩者都只是「過夜」，而非「長」住。 胡志強早已爆料，馬英九搞「郎史黛」是心不甘情不願。那很合常情：台灣居住條件有限，要一個蓋三十四年被子的人去睡別人的床，用別人的新被子，必然心裡「毛毛的」，不得安眠。 但這次選情告急，苦無正當步數，只好重操舊業，易名「荷馬史黛」，繼續玩「過夜」的遊戲。 如「過夜」表示親民，何必等到選舉才過夜？歷任總統候選人，不論藍綠，沒有一個需要靠「過夜」來證明他親民；美國總統會邀金主、友人到白宮睡林肯大床，卻不會去幹夜宿陌生人家這種事。 如果過夜目的在於「瞭解」民情，那他總統是白混了。平時不燒香，選前要在「睡覺時間」瞭解問題，豈非怪事？他的問題出在只知作秀，不願面對現實，無心也無力思考因應對策。 台灣是一日生活圈，馬英九用專機趕場拜組頭，豈有不能飛回家睡覺之理？硬要在別人家裡聊個天，洗個澡，睡一覺，只暴露他自命「皇恩浩蕩」的騙票心態。 馬英九在外過夜，「居心叵測」，不知道他如何向周美青請假？他的理由是否真的「表裡如一」？ （作者王景弘，資深新聞工作者）
