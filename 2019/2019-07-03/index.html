自由廣場》國民黨精英露餡比爛

July 3, 2019

◎ 王景弘

在黨國統治時代，國家安全領域排斥台灣人，國民黨無理壟斷，竟嘲諷台灣人「沒有國際觀」，缺乏國際事務人才。

老國民黨確有一批人才，為保障台灣安全籌謀定計，最大成就要算一九五四年的「中美共同防禦條約」，與一九七九年的「台灣關係法」，敵友分明站在民主陣營，安全受美國保障。

但日益赤化的國民黨新咖崛起，卻競相露餡比爛，唱衰兩蔣反共保台策略，不談「國家」安全，只扯血緣民族，把台灣當玩物，接受「一個中國」，不用國際政治語言談「兩岸關係」，只知在中國設定的框架內恐嚇自己。

這些看到中國高幹統戰眼神骨頭就酥軟的傢伙，警告台灣不要成為美國的「棋子」，但如果用棋局比喻，台灣也在下棋，兩蔣何嘗不是把美國當衛護台灣的「棋子」？

望眼二○二○，國民黨「人氣」最高的韓某，竟不知所云，稱台灣是「塞子」，如果他意思是要幫美、日陣營「塞」住中國進出太平洋通道，那還有道理，但進中聯辦的人意思絕非如此。

在中國特權經商致富的郭董，敵友不分，把大敵中國與保障台灣安全的美國混為一談，稱台灣是在美、中之間「走鋼索」，渾然不知兩蔣費盡心機一面倒，要爭取美國安全承諾。

染紅軍師充斥，自己白裡透紅的柯文哲，居然說國家主權意識是「Ｐ」話，他只知道用血緣、床頭床尾的兩性關係來表述與中國「關係」；說不清自己「和中」的主張，就指控民進黨「抗中」，還主張不理中國的敵意，管它會形同默認。

國民黨精英鬧笑話，要靠唱歌、搬「素還真」來接地氣，卻棄守最接地氣的「反併吞」、「反紅媒」、「反送中」，那只有等選民加上「反紅黨」！

（作者為資深新聞工作者）
