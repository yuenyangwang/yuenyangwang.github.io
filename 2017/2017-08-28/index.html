鏗鏘集》雕像也走向民主化

August 28, 2017

雕像用意在紀念人民心目中的英雄或「偉人」，但雕像不朽，「偉人」身價卻會因時代變遷而貶值，獨裁者的雕像，在民主浪潮下，更要走霉運。

東歐共黨領袖、伊拉克海珊雕像都被拉下當垃圾；蔣介石雕像在台灣學校、機關、公園、馬路圓環，無所不在，現在也遭到破壞或拆除的命運。

標榜包容與自由的美國，也正面臨南北戰爭時期，南軍英雄雕像去留的抗爭，川普態度曖昧，更招來混淆道德標準與「挑撥族群」的罪名。

美國雕像爭議不因獨裁，而因黑白種族紛擾。十九世紀，南方白人為保持黑奴制度，不惜脫離聯邦，引爆南北戰爭。南方戰敗，白人心有不甘，重建期一過，便大造南軍英雄雕像，並出現白人至上的三Ｋ黨。

時過境遷，一般人對這些雕像已經無感，維州還推荐南軍主帥與華盛頓同列國會的雕像堂。但活躍的黑人領袖和民主黨，認為這些雕像凸顯黑奴制度，不斷要求拆除，而白人種族主義者以「新納粹」興起，對抗日增。

南卡示威暴力是白人至上主義者所引起，川普談話卻顛三倒四，最後以不分是非，各打五十板引起公憤，猶太裔的白宮經濟顧問柯恩被迫公開批判川普，白宮企業顧問委員會成員紛紛出走。

新時代的抗爭，證明南軍英雄羅勃特．李有先見之明：一八六九年擬議建造蓋茨堡南北軍會戰紀念園時，他就主張不要再使傷口長存，而應讓內亂的印記消失。

近幾年，南方諸州處理雕像爭議，已經「民主化」，相信人民有權塑造他們所崇拜者的雕像，也有權把失去他們崇拜者的雕像，送進博物館或歷史垃圾。南卡暴力事件將加速民主化決定雕像的去留。（作者王景弘，資深新聞工作者）
