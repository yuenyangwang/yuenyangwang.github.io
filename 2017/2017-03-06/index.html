鏗鏘集》中華民國宣佈過獨立嗎？

March 6, 2017

台灣是一個舉世未見的獨特國家，它事實獨立於任何國家之外，在法理上，普遍也認為沒有「宣布獨立」的必要，但不需要「宣布獨立」的理由，表面相似，對台灣前途卻有重大差異。

改變憲法體制，成為第一位台灣民選總統的李登輝，以「主權在民」之說，認定台灣透過民主選舉已成為主權獨立國家；民進黨的台灣前途決議案，也主張主權在民，認定現狀就是主權獨立國家，未來改變需要二千三百萬人民決定。

但把外來憲法強加給台灣的統派，別有用心的混淆是非，另提不需要「宣布獨立」的理由，其經典之作就是馬英九在紐約的油嘴：你聽說過一個國家宣布獨立兩次嗎？「我們」早在一九一二年就是獨立國家了，我要怎麼第二次宣布獨立？

馬英九整段話是白賊，背離事實：菲律賓就「獨立」過兩次，第一次是從西班牙獨立，第二次是從美國獨立。

他所謂「我們」，如果指「台灣」，那一九一二年台灣是日本殖民地，絕非獨立國家；如果是指「中華民國」，那中華民國並未曾「宣布獨立」，領土也沒有台灣。

一九一二年的「中華民國」，外交照會只稱「推翻專制政府，建立民國」，那是成立新政府取代清朝，並不是宣布從大清帝國獨立；臨時政府外交總長王寵惠向美國也只要求儘快承認「我們的政府」。

中華人民共和國取代中華民國，也是「中國」的革命改朝換代，毛澤東不是「宣布獨立」，而是宣布「中華人民共和國中央政府成立」。

馬英九胡言牽拖的理由包藏禍心，故意無視舊金山和約未定台灣歸屬，挑戰台灣民主與主權在民，替中國對台灣的主權主張幫腔。（作者王景弘，資深新聞工作者）
