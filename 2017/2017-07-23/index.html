星期專論》外交部說帖的雲端與現實

July 23, 2017

◎王景弘

蔣介石政府代表二戰盟國受降及佔領台灣之後，用開羅公報、波茨坦宣言等戰時文件灌輸台、澎「歸還」中華民國的論調已逾七十年，現在教育部總算決定高中新課綱不再列入這套謬誤，這是教育求真求實的一大進步。

同時，外交部修訂「台灣地位」新說帖，雖仍錯誤及誇大解釋這些戰時文件，但承認那是外交部的見解，把這些歷史文件「視為」有條約效力的法律文件，因此，對日和約只是「再確認」台灣與澎湖歸還中華民國。

除了這些老掉牙的硬拗，外交部的說帖也從雲端回到現實，以台灣為主體，用民主鋪陳台灣的地位：第一，主權立論從佔領性質的「有效治理」，走到憲法規定的「主權在民」；第二，用憲法的「國民」來定位「中華民國」；第三，它強調開羅公告表明台澎等地應歸還「中華民國」，避免落入「中國」的圈套。

戰後有關台灣與澎湖主權問題的爭論，只有中共與蔣介石政權用開羅公報等文件，主張台灣與澎湖移交中國或中華民國，而不承認舊金山和約及「中日和約」未決定台灣與澎湖歸屬的法律事實。

國共堅稱開羅公告有約束力的立場，明顯誇張和錯誤。蔣介石在二戰期間未受盟國重視，備感屈辱，美國總統羅斯福怕他投降，為安撫他，才同意到開羅附近舉行「三巨頭」軍事會議。

因為會議主要談軍事問題，美方沒有國務院官員與會，蔣介石只帶一名外交部參事，英國則有外交部次官參加。被稱為「宣言」的文件，是臨時草擬，在會議結束，與會人員離開之後才宣佈的「新聞公報」（Press Communique）。

新聞公報內容及用語是三方同意，沒有簽字，秦孝儀編、國民黨黨史會出版的戰時外交文件，標題誠實地用「新聞公報」，而非「宣言」，駐美大使魏道明報告外交部，也只提白宮發表開羅會議「公告」要點。

美國把開羅公告及波茨坦宣言「視為」戰時盟國「意向聲明」，並非處理台澎主權的文件，這兩項文件也從未列入國務院的「有效條約彙編」。但因公告宣示要把台澎歸還「中華民國」，在內戰被推翻的國民黨政府，就因這句話主張開羅公告有約束力，而中共繼承國民黨政府，便堅持台澎已歸還「中國」。

但和約才是終結戰爭及處理戰敗國領土問題的法律文件。因盟國對於台澎領土應交給「中華民國」或「中華人民共和國」無法達成協議，決定在舊金山和約只規定日本宣佈「放棄」而不明定歸屬，這便造成台灣法律地位未定。

和約簽定後從未修改，也沒有任何國際會議或文件改變台灣主權，在理論上，台澎主權仍未最終處理。負責「中日和約」談判的外交部長葉公超，在立法院答詢也只能辯稱：「台澎原係中國領土。日本宣佈放棄後，只有中國有權接管。事實上，我們現在也控制台澎，毫無疑問地它構成我們領土的一部份。但是，微妙的國際情勢使它並不屬於我們所有。在目前情勢下，日本無權把台澎轉移給我們；即使他們願意，我們也不能接受轉移…。」

舊金山和約未定台澎歸屬，周恩來斥為「荒謬」，中國政府不承認，和約無效。他嘲笑蔣介石主張代表全中國，但卻被中國人民推翻，他現在（一九七一年）霸佔的地方（台澎）地位仍然未定，就此而言，蔣介石的政府是「吊在半空中」。

但台灣自民主化以來，除馬英九之外，都以台灣為主體，國家地位已定，並非停留在和約的「未定」狀態，也不是以長期有效佔領與統治主張取得主權，而是基於日本宣佈放棄主權，經長期民主演變，台澎主權屬於台灣人民。

外交部新說帖最大進步就在點出「主權在民」的論述：「中華民國」命運與台灣命運不可分，「中華民國」在台灣實行民主，國會全面改選及總統直選，「已將憲法主權在民的觀念徹底實現」。

依中華民國憲法，其主權屬於全國「國民」，而具有中華民國國籍者才是中華民國「國民」，這是以「國民」來定位「中華民國」，台澎兩千三百萬國民構成自己的國家。這種立論脫離雲端，以民主取代「法統」，雖尚未獲普遍正式承認，但立論更堅實和具說服力。

（作者王景弘，資深新聞工作者）
